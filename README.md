# AetherGuard Forensic | Deepfake Detection beyond Gravity (V5)

AetherGuard is a production-grade AI forensic suite designed for high-fidelity deconvolution of digital media. It utilizes an ensemble of EfficientNet-B4 and Vision Transformer architectures to identify biometric anomalies, optical misalignments, and generative artifacts in images, temporal video streams, and live optical sensors.

---

## 🚀 Key Forensic Capabilities:
*   **Multimodal Inference Core**: Unified analysis for Images, Video Streams, and Live Webcams.
*   **Ensemble Forensic Jury**: Step-by-step resolution of biometrics using MTCNN and Transformer-based feature extraction.
*   **Kinetic Log Stream**: Continuous, time-stamped scrolling marquee of forensic activities.
*   **Forensic Report Generation**: One-click generation of technical veracity documents.
*   **Vault Persistence**: Localized history storage for long-term investigative tracking.
*   **Enterprise UI (V5)**: Cinematic glassmorphism-based dashboard with real-time analytics (Volume, Fraud Ratio, Latency).

---

## 🛠️ Architecture:
*   **Frontend**: React, Tailwind CSS, Framer Motion, Lucide-Icons, Axios.
*   **Backend**: FastAPI, PyTorch, MTCNN, Transformers, Uvicorn.
*   **Optimization**: 3-tier state management with production-grade Error Boundaries.

## ⚡ Quick Start:

### 1. Backend Initialization:
```bash
# Navigate to the project root
cd backend
python -m venv venv
source venv/bin/activate # On Windows: .\venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

### 2. Frontend Deployment:
```bash
cd frontend
npm install
npm run dev
```

### 3. Engine Health:
Verify the engine link at `http://localhost:5173`. The **Engine Status** indicator must show `ONLINE`.
